### PROYECTOS DESTACADOS

**BarbeOS — Sistema de Gestión para Barberías**
<sub>Desarrollador Principal &nbsp;·&nbsp; 2024 – Actualidad</sub>

Aplicación de escritorio multiplataforma para la administración de turnos y servicios, con más de 15 versiones estables publicadas.
- Arquitectura de datos migrada de `localStorage` a **SQLite** para persistencia real.
- Implementación de seguridad mediante **Context Isolation** y comunicación IPC restringida.
- Automatización total del despliegue y empaquetado mediante pipelines de **GitHub Actions**.
