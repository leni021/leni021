## Proyectos

<br>

**BarbeOS — Sistema de Gestion para Barberias**

Sistema de escritorio multiplataforma para la administracion de turnos, clientes y servicios en barberias.

- Migracion de `localStorage` a **SQLite** para persistencia real de datos y soporte a esquemas relacionales
- Modelo de seguridad con **Context Isolation** y comunicacion restringida IPC entre procesos Main y Renderer
- Pipeline de **CI/CD con GitHub Actions**: empaquetado automatico de ejecutables y publicacion de releases sin pasos manuales
- **15+ versiones publicadas** — version estable en desarrollo activo
